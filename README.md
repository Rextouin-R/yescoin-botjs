# YESCOIN bot | via TERMUX 

## moduls by : maldiharyojudanto

## Install 
```
pkg install git
```
```
pkg install nano
```
```
pkg install nodejs
```
```
git clone https://github.com/Rextouin-R/yescoin-botjs/
```
## Buka file 
```
cd yescoin-botjs
```
```
npm install
```
## Edit `query.txt` lalu paste `query_id/user_id` kalian
## Untuk save `CTRL+xy` (enter) 
```
nano query.txt
```
## Run bot
```
node index.js
```
